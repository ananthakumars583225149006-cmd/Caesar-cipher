# 🔐 Caesar Cipher Tool

A simple, elegant web-based tool for encrypting and decrypting messages using the classic Caesar Cipher. Built as part of the VibeCoding Hackathon 6.0 sprint.

## 🎯 Features

- ✅ **Encrypt messages** with a custom shift value (1-25)
- ✅ **Decrypt messages** back to original text
- ✅ **Supports uppercase & lowercase** letters
- ✅ **Preserves symbols, numbers & spaces** (only letters shift)
- ✅ **Copy to clipboard** functionality
- ✅ **Keyboard shortcuts** (Ctrl+Enter to encrypt)
- ✅ **Dark theme UI** for comfortable use
- ✅ **100% client-side** - No data sent to any server
- ✅ **Mobile responsive** - Works on all devices

## 🛠️ Technologies Used

- **HTML5** - Structure
- **CSS3** - Styling & Animations
- **Vanilla JavaScript** - Logic & Interactivity
- **GitHub Pages** - Hosting

## 📖 How to Use

1. **Type your message** in the text area
2. **Choose a shift number** between 1-25
3. Click **🔒 Encrypt** to scramble your message
4. Click **🔓 Decrypt** to unscramble it back
5. Use the **📋 Copy** button to copy the result

### Example

| Input | Shift | Encrypt | Decrypt |
|-------|-------|---------|---------|
| Hello World | 3 | Khoor Zruog | Hello World |
| ABC | 1 | BCD | ABC |
| XYZ | 3 | ABC | XYZ |
| Hello123! | 5 | Mjqqt123! | Hello123! |

## ⌨️ Keyboard Shortcuts

- **Ctrl + Enter** - Quick encrypt the message
- **Enter** (in shift field) - Quick encrypt

## 🧠 How It Works

The Caesar Cipher is one of the simplest encryption techniques. Each letter in the plaintext is shifted by a fixed number of positions in the alphabet.
